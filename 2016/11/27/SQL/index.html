SQL增删改查基础语句

2016-11-27 • 标签: SQL, 基础

增加

INSERT INTO 表名（字段一，字段二，字段三）VALUES（”值1”,”值2”,”值3”）

删除

DELETE FORM 表名 WHERE 条件；

修改

UPDATE 表名 SET 设置的内容 加条件语句（WHERE 更改的内容）

查询

SELECT 字段 FORM 表名 WHERE 条件语句

联表查询

SELECT*FORM 表名1 JOIN 表名二 ON 条件
