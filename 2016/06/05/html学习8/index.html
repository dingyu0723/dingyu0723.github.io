2016-06-05 • 标签: html, 布局

css内容

.bigdiv{
    width: 400px;
    height: 400px;
    background: red;
    /*position: 
    定位，给一个元素通过定位改变布局的时候需要用到possition
    relative: 相对于一个自身的定位;
    */
    /*possition可以称之为完全脱离文档流，它的层级要比float的层级高，
    并且可以覆盖文字。*/
    /*position:relative;*/
    /*top: 100px;
    left: 100px;*/
    /*z-index:

    层级;*/
}
.topdiv{
    width: 300px;
    height: 100px;
    background: green;
    position: relative;
    /*float: left;*/
    z-index: 30;
}
.div1{
    width: 200px;
    height: 200px;
    background: blue;
    /*absolute：绝对定位，相对于父级
    absolute定位的时候会找带有possition属性的父级或者父父级
    （一直往上找），然后以带possition的父级来定位，如果都
    没有possition属性，则以左上角来定位。
    possition属性 可以设置top,left,bottom,right.
    只有写了的方向才会生效 ，可以为负数。*/
    position: absolute;
    top: 0px;
    z-index: 20;
}
body{
    height: 2000px;
}
.fixdiv{
    width: 100px;
    height: 400px;
    background: orange;
    /*fixed 固定定位
    始终在窗口的某一个位置。*/
    position: fixed;
    top: 100px;
    left: 0px;
}
